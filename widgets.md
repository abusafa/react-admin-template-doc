# Widget Foo

### Importing the widget module `FooWidget`.
> Widgets are found in the `./src/Widgets` folder.

```js
import FooWidget from './Widgets/FooWidget';
```

Using the `widget`
```html
<FooWidget
  prop1=""
  prop2=""
  prop3=""
  />

  <FooWidget
    prop1=""
    prop2=""
    prop3=""
  >
  ...
  </FooWidget>
```
