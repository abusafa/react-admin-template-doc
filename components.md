# Component Foo

### Importing the component module `FooComponent`.
> Components are found in the `./src/Components` folder.

```js
import FooComponent from './Components/FooComponent';
```

Using the `component`
```html
<FooComponent
  prop1=""
  prop2=""
  prop3=""
  />

  <FooComponent
    prop1=""
    prop2=""
    prop3=""
  >
  ...
  </FooComponent>
```
